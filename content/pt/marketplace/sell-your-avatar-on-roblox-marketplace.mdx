<Section icon={Map} title="Guia Passo a Passo para Publicação Direta">

Se você possui uma assinatura Roblox Premium, pode empacotar e publicar seus trajes diretamente pela interface de personalização do avatar. Este método registra o pacote de traje no banco de dados oficial da plataforma, tornando-o visível para os usuários que navegam pelos menus de personalização.

<Steps>
  <Step num="1" title="Monte o Traje">
    Equipe seu personagem com a combinação desejada de cabelo, acessórios faciais, pacotes e itens de vestuário personalizados.
  </Step>
  <Step num="2" title="Acesse a Ferramenta de Criação">
    Navegue até a página do Editor de Avatar. Localize a interface de criação e selecione a opção para salvar ou publicar a configuração atual do traje.
  </Step>
  <Step num="3" title="Defina os Metadados">
    Insira um nome descritivo e uma descrição detalhada para o seu traje. Use termos de pesquisa relevantes para melhorar a descoberta.
  </Step>
  <Step num="4" title="Finalize e Publique">
    Confirme a configuração. Se o traje contiver seus ativos personalizados, defina os parâmetros de preço e publique-o no marketplace.
  </Step>
</Steps>

Uma vez publicado, o traje é indexado. Outros jogadores podem visualizar o visual completo e comprar os componentes diretamente do seu perfil ou do marketplace.

| Componente da Listagem | Objetivo de Otimização | Impacto na Visibilidade |

| Descrição | Detalhar lista de itens e estilo | Média relevância de pesquisa |
| Preço | Manter itens individuais acessíveis | Alta taxa de conversão |

<Section icon={Target} title="Publicando sem Premium via Catalog Avatar Creator">

Para criadores que não possuem uma assinatura Premium, utilizar experiências especializadas dentro do jogo é uma alternativa altamente eficaz. A popular experiência **Catalog Avatar Creator** permite que os usuários criem, exibam e vendam trajes para uma comunidade massiva sem pagar taxas de assinatura antecipadas.

Este método baseia-se na geração de códigos de trajes exclusivos que outros jogadores podem inserir para carregar e comprar instantaneamente o estilo que você criou.

| Recurso | Método do Catálogo no Jogo | Marketplace Web Oficial |
| :--- | :--- | :--- |
| Premium Necessário | Não | Sim |
| Custo de Listagem | Grátis | 10 Robux por item de roupa |
| Compartilhamento Social | Alto via geração de código | Baixo via links de perfil |
| Alcance de Público | Jogadores ativos na experiência | Usuários de toda a plataforma |

To publicar usando este método, entre na experiência Catalog Avatar Creator e abra o menu de personalização. Personalize seu personagem, navegue até a seção de trajes da comunidade e clique na opção de publicar. Insira um título e uma descrição e, em seguida, envie o traje para gerar seu código de compartilhamento exclusivo.

<Callout type="info" title="Mecânica de Receita">
Quando os jogadores compram um traje no Catalog Avatar Creator usando o seu código, você recebe uma comissão de afiliado. Essa comissão é distribuída diretamente na sua conta como Robux pendentes.
</Callout>
